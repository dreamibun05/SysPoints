## CURRENT COMMANDS
**System management**
`sp_createsystem <system name>`
`sp_system`
`sp_addmember <name>`
`sp_members`
**Points management**
`sp_addpoints <member ID> <amount>`
`sp_removepoints <member ID> <amount>`
`sp_setpoints <member ID> <amount>`
`sp_checkpoints <member ID>`
`sp_givepoints @user <member ID> <amount>`
`sp_leaderboard`
**Chores**
`sp_chore add <points> <chore>`
`sp_chore list`
`sp_chore finish <chore ID> <member ID>`
`sp_chore delete <chore ID>`
***Danger zone!***
`sp_deletemember <member ID>`
`sp_purge`
*Note! Both sp_help and sp_commands work to access this menu at any time!*
